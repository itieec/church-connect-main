# ADR-002 — Roles Are Permission Templates

**Status:** Approved (Architecture Handbook v0.3)

## Decision

Roles are permission templates, not final authority. Default deny and least privilege. Explicit deny overrides grants. Assignments are scope-aware and time-aware when configured.

## Clarifications

See ADR-RBAC-001 / ADR-RBAC-002 / ADR-RBAC-003 for scoped assignment, live templates, default full-template grant, and time bounds.
