# ADR-003 — One Shared Operational Calendar Per Organization

**Status:** Approved (Architecture Handbook v0.3)

## Decision

Each organization uses one shared operational calendar. Ministries, teams, and groups create events inside that calendar under permission and conflict rules.

## Consequences

- Follow-Up Saturday program attendance links to this calendar
- Parent events may affect selected organizations only via explicit scope and conflict policy
- Parent events must not silently delete local events
